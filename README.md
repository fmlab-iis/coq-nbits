
This is a Coq library for bit vectors. A bit vector is represented as
`bitseq` (i.e. `seq bool`) with LSB at the head and MSB at the last.


Requirements
============

- Coq (>= 8.13.1)
- coq-mathcomp-ssreflect, coq-mathcomp-algebra (>= 1.12.0)
